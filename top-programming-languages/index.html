Top programming languages in use 2019

yourbasic.org
  • Language trends 2000–2019
  • Top languages 2019

This informal study gives a broad estimate of what programming languages are used commercially in the US. Starting in 2000, I’ve done a yearly job search at www.dice.com for some of the most well-known languages.

Here are some main takeaways:

  • You can clearly see the dramatic effects of the dot-com bubble in 2000.
  • It’s evident that some much-talked-about languages (let’s not mention any names) seem to find very little use in industry.
  • Java, JavaScript, and Python have taken a large and stable market share during the last 20 years.
  • Visual Basic has made a recent comeback.
