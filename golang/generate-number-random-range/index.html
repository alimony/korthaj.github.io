Generate a random number in a given range

yourbasic.org/golang

Use the rand.Seed and rand.Intn functions in package math/rand to generate a random number between a and b.

rand.Seed(time.Now().UnixNano())
n := a + rand.Intn(b-a+1)
Warning: Without the call to rand.Seed you will get the same sequence of pseudo­random numbers each time you run the program.
