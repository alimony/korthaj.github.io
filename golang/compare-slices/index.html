Compare slices (arrays)

yourbasic.org

In general, the preferred approach is to write your own code for comparing slices.

// Equal tells whether a and b contain the same elements.
// A nil argument is equivalent to an empty slice.
func Equal(a, b []int) bool {
    if len(a) != len(b) {
        return false
    }
    for i, v := range a {
        if v != b[i] {
            return false
        }
    }
    return true
}

Byte slices

For comparing bytes slices, use the optimized bytes.Equal. This function also treats nil arguments as equivalent to empty slices.

reflect.DeepEqual

For testing purposes, you may want to use reflect.DeepEqual.

var a []int = nil
var b []int = make([]int, 0)
fmt.Println(reflect.DeepEqual(a, b)) // false

The performance of this function is much worse than for the code above, but it’s useful in test cases where simplicity and correctness are crucial. The semantics, however, are quite complicated.
