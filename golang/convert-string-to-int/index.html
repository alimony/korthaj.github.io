Convert string to int

yourbasic.org/golang

Use strconv.Atoi to convert/parse a string to an int.

str := "123"
if n, err := strconv.Atoi(str); err == nil {
    fmt.Println(n+1)
} else {
    fmt.Println(str, "is not an integer.")
}
// Output: 124
